# 寻根族谱管理系统 v2.0.0 版本发布

## 概述

本次版本升级聚焦于用户体验全面提升，引入全新中式典雅风格前端，并新增三大核心功能模块。

## 新增功能

### 🎨 全新前端 (frontend-v2)
- 中式传统典雅风格 UI 设计
- 自定义组件库（GCard、GButton、GInput），脱离 Element Plus 依赖
- 内联 SVG 图标体系，CSS 变量驱动主题
- 家族详情页多标签页直接展示内容，无需二次跳转

### 📸 家族相册
- 相册集管理：创建多个相册分类管理家族照片
- 照片上传：支持标题、拍摄日期，自动生成缩略图
- 时间轴视图：按年份分组展示家族历史照片
- 照片信息编辑：支持上传后修改标题和拍摄日期

### 📜 文献库
- 文献上传：支持 PDF、图片、Word 等格式
- 分类管理：老谱扫描、契约文书、著作文献等
- 在线预览：图片和 PDF 内联预览
- 搜索筛选：按分类、关键词快速定位

### 📖 人物传记
- 独立传记页面：为家族名人创建专属传记
- 富文本编辑：支持 HTML 格式正文
- 浏览量统计：自动记录浏览次数
- 传记管理：创建、编辑、发布/草稿控制

### 🏆 功德榜
- 修谱捐款记录：记录宗亲捐款信息
- 金额排行：按捐款金额降序排列，前三名奖牌标识
- 数据统计：捐款人次和总额一目了然

## 功能优化

- 照片标题默认"未命名"，上传后支持编辑
- 族谱树 tab 自动以第一代成员为根节点加载，无需手动选择
- 各功能 tab 直接展示核心内容，减少操作步骤
- 移动端适配优化：全局响应式断点、表格卡片化、触摸目标优化

## 技术变更

- **后端新增模型**：Album、Photo、Document、PersonBiography、MeritDonor
- **后端新增 API**：相册（10 端点）、文献（6 端点）、传记（7 端点）、功德榜（5 端点）
- **前端双版本**：v1（Element Plus 旧版）+ v2（自定义组件新版，主推）
- **文件上传扩展**：支持 PDF/Word 等文档类型
- **配置扩展**：新增文档上传大小限制、允许类型配置
